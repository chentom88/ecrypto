# Ecrypto

Just me messing around with some very basic crypto stuff for education.

## Simple Setup

```
go get github.com/chentom88/ecrypto
```

## Development Setup

- Install glide 

```
curl https://glide.sh/get | sh
```
- Install dependencies
```
glide install
```
- Run build
```
scripts/build.sh
```

## Run
- After building
```
./ecrypto --help
```

## Running Test
- Install [Gingko](https://github.com/onsi/ginkgo)
- Run `scripts/test.sh`
